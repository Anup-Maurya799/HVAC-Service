# AMSUNO HVAC Website

A modern and responsive HVAC service website built using **React**, **Vite**, and **Tailwind CSS**. The website showcases HVAC services, company information, customer testimonials, and a contact section with a clean and user-friendly interface.

## Features

- Responsive design for all devices
- Modern and clean UI
- Sticky navigation bar
- Mobile-friendly navigation menu
- Hero section with call-to-action buttons
- Services section
- About company section
- Customer testimonials
- Contact form
- Smooth hover animations
- Footer with social media links

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Icons
